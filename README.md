Incredibly quickly simulate a keypress on MacOS


## Install

###### [Homebrew](https://brew.sh)

```sh
brew install keysim
```

###### Manually

[Download](https://github.com/soheil/keysim/releases/latest) the binary and put it in `/usr/local/bin`.

## Usage

```sh
keysim x
```
